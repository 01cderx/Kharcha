#Expense Tracker
# 💸 Expense Tracker WebApp (MERN Stack)

A basic but functional **Expense Tracker** built using the **MERN stack (MongoDB, Express, React, Node.js)**.  
It allows users to add, view, and delete their expenses — with full backend integration and MongoDB database support.

## 🚀 Features

- ➕ Add new expense with amount & description
- 🗑️ Delete specific expenses
- 📊 View all expenses in a list
- 🔄 Real-time data updates (via API calls)
- 🗃️ Data stored in **MongoDB**
- 🌐 Frontend built in **React**
- 📦 Backend built in **Node.js + Express**

---

## 🛠️ Tech Stack

**Frontend:**
- React
- JavaScript
- Axios or Fetch API
- Basic CSS for UI

**Backend:**
- Node.js
- Express.js
- MongoDB (with Mongoose)
